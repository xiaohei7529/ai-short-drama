## 📖 剧本模板

### 1. 霸道总裁爱上我 (共 5 集)

**类型**: 都市言情  
**时长**: 每集 2-3 分钟  
**角色**: 2 主角 + 3 配角

#### 角色设定
- **顾北辰** (男主): 28 岁，顾氏集团总裁，高冷帅气，内心温柔
- **苏浅浅** (女主): 24 岁，普通白领，善良乐观，有点小迷糊
- **林婉儿** (女配): 26 岁，顾北辰未婚妻，富家千金，心机深
- **王特助** (男配): 30 岁，顾北辰特助，忠诚能干
- **张妈** (女配): 50 岁，顾家保姆，慈祥热心

#### 分集大纲

**第 1 集: 意外相遇**
- 场景：咖啡厅
- 情节：苏浅浅送外卖不小心撞到顾北辰，咖啡洒了一身
- 冲突：顾北辰要求赔偿，苏浅浅发现他是新老板
- 悬念：顾北辰对苏浅浅产生兴趣

**第 2 集: 职场重逢**
- 场景：公司办公室
- 情节：苏浅浅入职发现顾北辰是总裁
- 冲突：林婉儿刁难苏浅浅
- 悬念：顾北辰暗中保护苏浅浅

**第 3 集: 误会加深**
- 场景：宴会厅
- 情节：林婉儿设计陷害苏浅浅
- 冲突：顾北辰误会苏浅浅
- 悬念：苏浅浅伤心离开

**第 4 集: 真相大白**
- 场景：医院
- 情节：苏浅浅救顾北辰受伤
- 冲突：林婉儿阴谋败露
- 悬念：顾北辰向苏浅浅表白

**第 5 集: 幸福结局**
- 场景：婚礼现场
- 情节：顾北辰和苏浅浅举行婚礼
- 冲突：林婉儿最后捣乱被赶走
- 结局：有情人终成眷属

## 🎯 使用方法

### 方式一：直接使用
1. 选择剧本模板
2. 复制剧本内容
3. 在系统中上传
4. AI 自动生成分镜和视频

### 方式二：自定义修改
1. 参考剧本结构
2. 修改角色设定
3. 调整情节走向
4. 生成个性化短剧

### 方式三：AI 生成
1. 输入故事创意
2. 选择剧本类型
3. AI 自动生成完整剧本
4. 人工审核调整

## 📝 剧本格式规范

```markdown
# 剧本标题

**类型**: 类型名称
**集数**: X 集
**时长**: 每集 X 分钟

## 角色设定
- **角色名** (性别): 年龄，职业，性格特点

## 分集大纲

### 第 X 集：标题
- 场景：地点
- 情节：故事发展
- 冲突：矛盾点
- 悬念：吸引点
```

## 💡 创作技巧

1. **开头抓人** - 前 3 秒必须吸引观众
2. **节奏紧凑** - 每 30 秒一个转折
3. **冲突明显** - 制造戏剧张力
4. **悬念结尾** - 引导观众看下一集
5. **情感共鸣** - 触动观众内心
